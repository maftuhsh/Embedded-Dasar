# 📚 Embedded Systems Basic Practicum

Repositori ini berisi kumpulan kode dan materi dari praktikum Embedded Systems Basic. Semua modul disusun secara terstruktur supaya mudah dipelajari, baik untuk pemula maupun yang ingin memperdalam konsep embedded.

## 🎯 Tujuan

Repositori ini dibuat untuk:

Mendokumentasikan hasil praktikum
Memudahkan proses belajar secara mandiri
Menjadi referensi bagi mahasiswa lain

## 🗂️ Struktur Folder

Setiap folder mewakili satu modul praktikum, yang berisi:

Source code
Penjelasan singkat
(Opsional) rangkaian atau dokumentasi tambahan

## 🚀 Cara Pakai
Clone repositori ini
Pilih modul yang ingin dipelajari
Baca penjelasan singkatnya
Jalankan kode dan coba modifikasi sendiri\

## 💡 Tips Belajar
Jangan cuma copy-paste, pahami alurnya
Eksperimen dengan mengubah parameter
Coba gabungkan beberapa modul jadi project kecil

## 🤝 Kontribusi

Kalau kamu punya ide, perbaikan, atau tambahan modul, feel free untuk kontribusi lewat pull request!

## 📌 Catatan

Kode di sini dibuat untuk tujuan pembelajaran, jadi mungkin belum sepenuhnya optimal atau production-ready.
