# 🚀 Modul 2 Embedded Dasar
Pemrograman Sensor Gas & Motor DC
## 📌 Deskripsi 

Proyek ini merupakan implementasi sistem embedded berbasis Arduino yang mengintegrasikan sensor gas (MQ-5) dengan motor DC menggunakan driver L298N.

Sistem dirancang untuk:

Membaca konsentrasi gas dari lingkungan
Mengolah data tersebut
Mengontrol aktuator (motor DC) berdasarkan kondisi tertentu

Secara konseptual, ini adalah contoh closed-loop system sederhana di mana input (gas) mempengaruhi output (motor).

## 🎯 Tujuan
Memahami pembacaan sensor analog
Mengontrol arah & kecepatan motor DC
Mengimplementasikan logika kontrol berbasis kondisi
Mengintegrasikan sensor + aktuator dalam satu sistem

## 🔧 Komponen yang Digunakan
Arduino UNO
Sensor Gas MQ-5
Motor DC
Motor Driver L298N
Push Button
LED (indikator)
Breadboard & Kabel Jumper
